# Описание функций 

В этом документы описаны все функциии, реализованные в проекте.

 

### `area(r)`

Возвращает площадь круга радиуса r.
    
**Параметры:**
- r(int/float): радиус окружности
    
**Возвращаемое значение:**
- area(int/float): Произведение π и r и r

**Пример:**
```python
result = area(5)
print(result) # Вывод: 78.53981633974483
```

### `perimeter(r)`

Возвращает периметр(длину окружности) радиуса r.

**Параметры:**
- r(int/float): радиус окружности
        
**Возвращаемое значение:**
- perimeter(int/float): произведение 2 и π и r
        
**Пример:**
```python
result = perimeter(5)
print(result) # Вывод: 31.41592653589793
```

## Модуль `rectangle`

### `area(a,b)`

Находит площадь прямоугольника со сторонами a и b.

**Параметры:**
- a(int/float): длина прямоугольника
- b(int/float): ширина прямоугольника
        
**Возвращаемое значение:**
- area(int/float,int/float): произведение a и b 
        
**Пример:**
```python
result = area(3,4)
print(result) # Вывод: 12
```
### `perimeter(a,b)` 

Находит периметр прямоугольника со сторонами a и b.

**Параметры:**
- a(int/float): длина прямоугольника
- b(int/float): ширина прямоугольника
        
**Возвращаемое значение:**
- area(int/float,int/float): сумма a и a и b и b 
        
**Пример:**
```python
result = area(3,4)
print(result) # Вывод: 14
```

## Модуль `square`

### `area(a)`

Находит площадь квадрата со стороной a.

**Параметры:**
- a(int/float): сторона квадрата

**Возвращаемое значение:**
- area(int/float): произведение a и a 
        
**Пример:**
```python
result = area(3)
print(result) # Вывод: 9
```
### `perimeter(a)` 

Находит периметр квадрата со стороной a.

**Параметры:**
- a(int/float): сторона квадрата
        
**Возвращаемое значение:**
- area(int/float,int/float): произведение 4 и a
        
**Пример:**
```python
result = area(4)
print(result) # Вывод: 16
```

## Модуль `triangle`

### `area(a, h)`

Возращает площадь треугольника с основанием a и высотой h

**Параметры:**
- a(int/float): основание треугольника
- h(int/float): высота треугольника

**Возвращаемое значение:**
- area(int/float): произведение a и h и 1/2
        
**Пример:**
```python
result = area(3,4)
print(result) # Вывод: 6
```
### `perimeter(a, b, c)` 

Возращает периметр треугольника со сторонами a, b, c.

**Параметры:**

- a(int/float): 1-ая сторона треугольника
- b(int/float): 2-ая сторона треугольника
- c(int/float): 3-ая сторона треугольника
        
**Возвращаемое значение:**
- perimeter(int/float,int/float): сумма a, b, c
        
**Пример:**
```python
result = perimeter(3,4,5)
print(result) # Вывод: 12
```
