# Цели и задачи тестирования

Цель: верификация корректности вычислений показателей геометрических фигур на корретных данных

Задачи:
- Проверка корректности математических вычислений,
- Подтверждение ожидаемого поведения при вводе неположительных чисел.

# Описание тестируемого продукта

geometric_lib — мини-библиотека с реализацией геометрических расчётов, написанная на Python 3.x (площадь и периметр квадрата, прямоугольника, треугольника, круга).

# Область тестирования

Функции для расчета геометрических показателей для каждой фигуры: area(), perimeter()

# Стратегия тестирования

Тип тестирования: модульное

Техника: Black-box тестирование

Фреймворк: unittest

Запуск: командная строка: python3 -m unittest tests/test_name.py

# Критерии приемки

Продукт считается прошедшим тестирование при выполнении следующих условий:
- Все тесты выполнены с результатом OK.
- Не наблюдается аварийных завершений.
- Вычисления совпадают с ожидаемыми значениями.
